Actividad de Reconocimiento

1.Configurar una cuenta de usuario en la plataforma www.github.com

2.Instalar el cliente de GIT para escritorio en su computador.

3.Instalar el cliente de GITHUB para escritorio en su computador.

4.Instalar el cliente de GIT para escritorio en su computador.

5.Crear un repositorio de prueba el cual debe estar sincronizado con el cliente de escritorio y su cuenta local de GITHUB.

6.Crear un archivo con el nombre index.html y lo subirá desde su computador al repositorio local de GITHUB para luego sincronizarlo con el remoto (www.github.com).

7.Publicar mediante documento PDF el link de su repositorio creado en GITHUB. Para su respectiva revisión.
